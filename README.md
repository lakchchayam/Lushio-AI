# Lushio AI - Multi-Agent Store Assistant

Professional-grade AI assistant featuring Multi-Agent architecture and RAG integration.

## 📁 Structure
- `src/`: Core backend logic (`agent.py`, `inventory.json`).
- `frontend/`: Chat UI.
- `tests/`: Evaluation scripts.
- `docs/`: Knowledge base (`faq.txt`).

## 🚀 Run
1. `python src/agent.py`
2. Open `frontend/index.html`
